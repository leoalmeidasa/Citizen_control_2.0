## Setup

1. `git clone https://github.com/leoalmeidasa/Citizen_control
2. `cd Citizen_control`
3. `rake db:create db:migrate`
4. `rspec spec/` (para verificar se todos os testes estão ok)
5. `rails s`
6. Abrir o link localhost:3000
